    class Dice_Stats::Internal_Utilities::Filtered_distribution

    This class is used as a way to build clauses into a query of a probability distribution The probability_distribution start out complete and each query removes selections of it.

    eq(val) click to toggle source

    The “equal to” operator. Note: This removes all options except the one specified. This is not useful in conjunction with any other operators. The result of “p.eq(x).eq(y).get” will always be 0 if x and y are different.

    # File lib/Internal_Utilities/Filtered_distribution.rb, line 56
def eq(val)
        @pd.select! { |k,v| k == val }
        return self
end
